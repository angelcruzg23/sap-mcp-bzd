¿Qué estás viendo?

🔐 Hash

Cada bloque tiene una "huella digital" única calculada a partir de sus datos. Si cambias algo, el hash cambia completamente.

⛓️ Encadenamiento

Cada bloque guarda el hash del bloque anterior. Esto crea una cadena donde alterar un bloque rompe todos los siguientes.

🛡️ Inmutabilidad

Edita los datos de cualquier bloque y observa cómo la cadena se "rompe". Así es como blockchain detecta manipulaciones.
