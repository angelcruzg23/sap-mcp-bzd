## 📦 Instrucciones Rápidas

### 1. Clonar el repositorio
```powershell
cd C:\Users\$env:USERNAME
git clone https://github.com/angelcruzg23/sap-mcp-bzd.git
cd sap-mcp-bzd
```

### 2. Ejecutar el asistente
```powershell
.\setup-wizard.ps1
```

El asistente te guiará paso a paso:
- Información personal (nombre, email, equipo)
- Selección de sistemas SAP (BZD para ti)
- Credenciales (usuario y password)
- Preferencias de trabajo

### 3. Reiniciar Kiro

Cerrar y abrir Kiro nuevamente.

### 4. Verificar

En Kiro, escribe:
```
Verifica la conexión con SAP BZD
```
